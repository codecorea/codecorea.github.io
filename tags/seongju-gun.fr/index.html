Un moment très spécial passé à Hangae Village - Seongju-gun, Gyeongbuk, Corée du Sud

Le village de Seongju Hangae, avec ses 600 ans d'histoire, conserve son apparence ancienne, mais n'est pas aussi bruyant ou occupé que le village de Hahoe et le village de Yangdong. Parmi eux, la maison de Jinsa est le seul endroit où vous pourrez découvrir un manoir dans le village, et vous pourrez déguster une table champêtre de bien-être. La maison de Jinsa est le meilleur endroit pour passer un moment tranquille dans le village de Hangae. Seongju-gun, Gyeongbuk, Corée du Sud. Korea Tour
